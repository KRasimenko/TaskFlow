# Руководство пользователя TaskFlow

## Запуск

1. Запустите сервер: `cd taskflow-server && mvn spring-boot:run`
2. Установите приложение из `taskflow-app`

## Основные функции

| Экран | Действие |
|-------|----------|
| Вход / Регистрация | Создание аккаунта на сервере |
| Главная | Список задач, поиск, фильтры |
| «+» | Создание задачи |
| Календарь | Задачи по датам |
| Избранное | Задачи со звездой |
| Профиль | Данные пользователя |

## Офлайн

При отсутствии сети данные сохраняются локально (SharedPreferences) и синхронизируются при восстановлении связи.
