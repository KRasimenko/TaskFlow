# Техническое задание (краткая версия)

## 1. Наименование

Разработка мобильного приложения для планирования задач (TaskFlow).

## 2. Основание

Курсовой проект по дисциплине «Программная инженерия», траектория В.

## 3. Назначение

Автоматизация личного и рабочего планирования задач.

## 4. Требования к функциям

- F1: Регистрация и вход (JWT)
- F2: CRUD задач
- F3: Поиск, фильтры, избранное
- F4: Календарь и уведомления
- F5: Профиль пользователя
- F6: REST API 8+ методов
- F7: Офлайн-кэш на клиенте

## 5. Нефункциональные требования

- Android 10+ (API 29)
- Время отклика UI < 200 мс
- Покрытие тестами сервера > 40 %
- Документация docs/01–08

## 6. Состав работ

См. WBS и диаграмму Ганта в `docs/plantuml/`.

## 7. COCOMO (упрощённая оценка)

- Размер: ~8 KLOC → Organic mode
- Effort ≈ 2.4 × (8)^1.05 ≈ 22 чел.-мес. (учебный проект ~ 2 мес.)
