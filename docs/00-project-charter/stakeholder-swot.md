# Матрица стейкхолдеров

| Стейкхолдер | Интерес | Влияние | Коммуникация |
|-------------|---------|---------|--------------|
| Студент (разработчик) | Защита курсового проекта | Высокое | Ежедневно |
| Руководитель (Свистунов И.В.) | Соответствие МУ, качество ПЗ | Высокое | Еженедельно |
| Проверяющий (комиссия) | Соответствие траектории В | Высокое | На защите |
| Конечный пользователь | Удобное планирование задач | Среднее | UAT, отзывы |
| Администратор сервера | Стабильность API и БД | Среднее | По регламенту |

## SWOT-анализ

| | Помогает | Мешает |
|---|----------|--------|
| **Внутреннее** | **S:** Kotlin, Spring Boot, готовый UI | **W:** Ограниченный бюджет, один разработчик |
| **Внешнее** | **O:** Спрос на task-менеджеры | **T:** Конкуренция (Todoist, Google Tasks) |

**S (Strengths):** открытый стек, модульная архитектура PCMEF, офлайн-кэш.  
**W (Weaknesses):** нет push-уведомлений FCM, начальная версия без облачной синхронизации в реальном времени.  
**O (Opportunities):** интеграция с календарём Google, виджеты.  
**T (Threats):** изменения политики Google Play, требования GDPR.
