## Executive Summary

TaskFlow — клиент-серверная система для управления личными и рабочими задачами. Android-клиент (Kotlin) взаимодействует с REST API (Spring Boot, JWT). Локальный кэш SharedPreferences обеспечивает офлайн-режим.

## Цели

- Централизованное хранение задач на сервере
- Мобильный UI по макету Figma
- Аутентификация JWT
- Покрытие тестами серверной части > 40 %
