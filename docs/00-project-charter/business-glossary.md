# Бизнес-глоссарий (15+ терминов)

| № | Термин | Определение |
|---|--------|-------------|
| 1 | Задача (Task) | Единица планирования с названием, описанием, сроком и статусом |
| 2 | Дедлайн | Крайний срок выполнения задачи (dueDate) |
| 3 | Приоритет | Уровень важности: низкий, средний, высокий |
| 4 | Статус | Состояние задачи: к выполнению, в работе, готово |
| 5 | Категория | Тематическая группа задачи (работа, личное и т.д.) |
| 6 | Избранное | Задачи, отмеченные звёздочкой (starred) |
| 7 | Календарь | Представление задач по датам |
| 8 | Уведомление | Напоминание о просроченных или текущих задачах |
| 9 | Пользователь | Аутентифицированный владелец задач |
| 10 | JWT | JSON Web Token для безопасной авторизации API |
| 11 | REST API | Интерфейс обмена данными клиент–сервер |
| 12 | Офлайн-режим | Работа с локальным кэшем без сети |
| 13 | PCMEF | Presentation–Control–Entity–Mediator–Foundation |
| 14 | CRUD | Create, Read, Update, Delete — базовые операции |
| 15 | SharedPreferences | Локальное хранилище ключ–значение на Android |
| 16 | Spring Boot | Java-фреймворк для серверной части |
| 17 | Use Case | Сценарий взаимодействия пользователя с системой |
